# 📊 RetailAB Sales Analytics | SQL Server & Power BI

## 🚀 Project Overview

RetailAB Sales Analytics is an end-to-end Business Intelligence project developed using **SQL Server** and **Power BI** to analyze retail sales performance, customer behavior, profitability, regional trends, and shipping operations.

The project follows a complete data analytics workflow, starting from data storage and management in SQL Server to advanced dashboard creation in Power BI. Interactive reports provide valuable business insights that support data-driven decision-making.

---

## 🎯 Business Objective

The primary objective of this project is to help business stakeholders:

* Monitor overall sales performance
* Track profitability across products and categories
* Analyze regional and state-wise sales trends
* Understand customer purchasing behavior
* Evaluate shipping and delivery performance
* Identify top-performing and underperforming products
* Support strategic business decisions through interactive reporting

## 📊 Dataset Overview

The project uses nine interconnected tables:

* Address
* Customers
* Date Table
* Products
* Regions
* Returns
* Sales
* Segment
* Ship Mode

These tables were imported into SQL Server and modeled in Power BI using relationships and dimensional modeling techniques.

---

## 🏗️ Data Model Highlights

* Star Schema Design
* Date Dimension Implementation
* Active & Inactive Relationships
* Product Hierarchy
* Customer Geography Hierarchy
* Fiscal Year & Fiscal Quarter Calculations
* DAX Measures for KPI Tracking
